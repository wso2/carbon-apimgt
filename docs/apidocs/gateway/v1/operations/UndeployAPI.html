Undeploy API

Uneploy the API in the gateway

POST https://apis.wso2.com/api/am/gateway/v1/undeploy-api

This operation is used to undeploy an API in the gateway.

OAuth 2.0 Scope

apim:api_publish

Examples

apiId

required

**API ID** consisting of the **UUID** of the API. The combination of the provider of the API, name of the API and the version is also accepted as a valid API I. Should be formatted as **provider-name-version**.

String
