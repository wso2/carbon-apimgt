Get API Artifact

Get API artifact from the storage

GET https://apis.wso2.com/api/am/gateway/v1/api-artifact

This operation is used to get the API artifact from the storage

OAuth 2.0 Scope

apim:api_publish

Examples

Parameters

Type Name Description Schema Default
Query

apiName

required

Name of the API

String

Query

label

required

Label of the API Gateway

String

Query

apiId

required

**API ID** consisting of the **UUID** of the API. The combination of the provider of the API, name of the API and the version is also accepted as a valid API I. Should be formatted as **provider-name-version**.

String

Responses

HTTP Code Description Schema

200

OK. API artifact successfully retrieved from the storage.

DeployResponse

202

Accepted. The request has been accepted.

DeployResponse

400

Bad Request. Invalid request or validation error

Error

404

Not Found. Requested API does not exist.

Error

412

Precondition Failed. The request has not been performed because one of the preconditions is not met.

Error
