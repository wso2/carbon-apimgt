Comments

Add an API comment

POST https://apis.wso2.com/api/am/store/v1.0/apis/{apiId}/comments

OAuth 2.0 Scope

apim:subscribe

Examples

Request
POST https://localhost:9443/api/am/store/v1.0/apis/b72b00c0-af2a-450a-a2e8-fbb526cf1ae8/comments HTTP 1/1 Authorization: Bearer eff01a97-4dc0-3542-a184-75aecf355ec2 Content-Type: application/json { "content" : "This is a comment" }
Response
HTTP 1/1 200 OK Content-Type: application/json { "id" : "c15901c7-407a-43ca-9cf9-d1f0c17f689d", "content" : "This is a comment", "createdTime" : "2019-09-03 14:27:47.02", "createdBy" : "john" }
CURL
curl -k -v -X POST 'https://localhost:9443/api/am/store/v1.0/apis/b72b00c0-af2a-450a-a2e8-fbb526cf1ae8/comments' -H 'Authorization: Bearer eff01a97-4dc0-3542-a184-75aecf355ec2' -H 'Content-Type: application/json' -d @payload.json
