Alerts

Get the list of API Store alert types.

GET https://apis.wso2.com/api/am/store/v1.0/alert-types

This operation is used to get the list of supportd alert types for the 'subscriber' agent.

OAuth 2.0 Scope

apim:sub_alert_manage

Examples

Request
GET https://localhost:9443/api/am/store/v1.0/alert-types HTTP 1/1 Authorization: Bearer eff01a97-4dc0-3542-a184-75aecf355ec2
Response
HTTP 1/1 200 OK Content-Type: application/json { "count" : 2, "alerts" : [ { "id" : 3, "name" : "AbnormalRequestsPerMin", "isRequireConfiguration" : true }, { "id" : 4, "name" : "AbnormalRequestPattern", "isRequireConfiguration" : false } ] }
CURL
curl -k -v -X GET 'https://localhost:9443/api/am/store/v1.0/alert-types' -H 'Authorization: Bearer eff01a97-4dc0-3542-a184-75aecf355ec2'

Parameters

Type Name Description Schema Default

Responses

HTTP Code Description Schema

200

OK. The list of subscriber alert types are returned.

AlertTypesList
