Download an API specific mediation policy

GET https://apis.wso2.com/api/am/publisher/v1/apis/{apiId}/mediation-policies/{mediationPolicyId}/content

This operation can be used to download a particular API specific mediation policy.

OAuth 2.0 Scope

apim:api_view apim:mediation_policy_view

Examples

retrieve content of a mediation policy of an api

Request
GET https://localhost:9443/api/am/publisher/v1/apis/a8f0fb89-edf9-484c-b937-036b088f287d/mediation-policies/624b9f7d-bfaf-484b-94cc-e84491f5d725/content HTTP 1/1 Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8
Response
HTTP 1/1 200 OK Content-Disposition: attachment; filename="sampleSeq.xml" Content-Type: application/xml Content-Length: 166
