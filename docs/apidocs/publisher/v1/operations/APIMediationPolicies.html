API Mediation Policies

Get all mediation policies of an API

GET https://apis.wso2.com/api/am/publisher/v1/apis/{apiId}/mediation-policies

This operation provides you a list of available mediation policies of an API.

OAuth 2.0 Scope

apim:api_view apim:mediation_policy_view

Examples

get mediation policies of an api

Request
GET https://localhost:9443/api/am/publisher/v1/apis/a8f0fb89-edf9-484c-b937-036b088f287d/mediation-policies HTTP 1/1 Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8
Response
HTTP 1/1 200 OK Content-Type: application/json { "id" : "24cea00d-9f4a-48d5-89b0-cbf01d487782", "name" : "header_sequence1", "type" : "IN", "sourceType" : "FILE" }
CURL
curl -k -v -X GET 'https://localhost:9443/api/am/publisher/v1/apis/a8f0fb89-edf9-484c-b937-036b088f287d/mediation-policies' -H 'Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8'

Add an API specific mediation policy

POST https://apis.wso2.com/api/am/publisher/v1/apis/{apiId}/mediation-policies

This operation can be used to add an API specifc mediation policy.

OAuth 2.0 Scope

apim:api_create apim:mediation_policy_create

Examples

Using a IN sequence as a file

Request
POST https://localhost:9443/api/am/publisher/v1/apis/a8f0fb89-edf9-484c-b937-036b088f287d/mediation-policies HTTP 1/1 Authorization: Bearer eff01a97-4dc0-3542-a184-75aecf355ec2 Content-Type: multipart/form-data; boundary=------------------------3da639a2c2734c24 Content-Length: 457
