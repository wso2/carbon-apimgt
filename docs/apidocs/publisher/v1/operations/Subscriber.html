Subscriber

Get details of a user who subscribed an API

GET https://apis.wso2.com/api/am/publisher/v1/subscriptions/{subscriptionId}/subscriber-info

This operation can be used to get details of a user who subscribed to the API.

OAuth 2.0 Scope

apim:subscription_view
