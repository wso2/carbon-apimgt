GraphQL Schema

Add a Schema to a GraphQL API

PUT https://apis.wso2.com/api/am/publisher/v1/apis/{apiId}/graphql-schema

This operation can be used to add a GraphQL Schema definition to an existing GraphQL API.

OAuth 2.0 Scope

apim:api_create
