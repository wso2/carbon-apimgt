External Resources (Collection)

Get all service endpoints after service discovery
GET https://apis.wso2.com/api/am/publisher/v1.0/external-resources/services

This operation can be used to retrieve the list of service endpoints available in the cluster after a process of service discovery.

OAuth 2.0 Scope

apim:external_services_discoveri

GET https://127.0.0.1:9292/api/am/publisher/v1.0/external-resources/services Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8

Response
