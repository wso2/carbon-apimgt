Remove Threat Protection Policy Association

Remove a policy accociated with an API
DELETE https://apis.wso2.com/api/am/publisher/v1.0/threat-protection/apis/{apiId}/policies/{policyId}

This can be used to remove a policy associated with an API

OAuth 2.0 Scope

apim:api_viewi

Request

DELETE https://apis.wso2.com/api/am/publisher/v1.0/threat-protection/apis/{apiId}/policies/{policyId}

Response

Not Available

Sample CURL

Not Available

Parameters

Type Name Description Schema Default
Path

apiId

required

**API ID** consisting of the **UUID** of the API. The combination of the provider of the API, name of the API and the version is also accepted as a valid API ID. Should be formatted as **provider-name-version**.

String

Path

policyId

required

The UUID of a Policy

String

Responses

HTTP Code Description Schema

200

Ok. Policy association removed.
