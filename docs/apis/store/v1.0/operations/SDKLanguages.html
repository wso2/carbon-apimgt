SDK Languages

Get a list of supported SDK languages
GET https://apis.wso2.com/api/am/store/v1.0/sdk-gen/languages

This operation will provide a list of programming languages that are supported by the swagger codegen library for generating System Development Kits (SDKs) for APIs available in the API Manager Store

OAuth 2.0 Scope

apim:subscribei

Request

GET https://localhost:9292/api/am/store/v1.0/sdk-gen/languages Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8

Response

HTTP/1.1 200 OK Content-Type : application/json

Sample CURL

curl -X GET -k -H "Authorization:Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8" https://localhost:9292/api/am/store/v1.0/sdk-gen/languages

Parameters

Type Name Description Schema Default

Responses

OK. List of supported languages for generating SDKs.
